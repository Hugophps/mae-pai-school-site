# Site - Sauvons l'ecole de Mae Pai

Site statique bilingue FR/EN pour la collecte de fonds et le volontariat autour de l'ecole de Mae Pai (Thailande).

## Lancer en local

Depuis le dossier du projet:

```bash
cd "/Users/hugophp/Documents/App Sans Concept/mae-pai-school-site"
python3 -m http.server 4173
```

Puis ouvrir `http://localhost:4173`.

## Deploiement gratuit

### Recommandation: Vercel

Pourquoi: deploiement le plus rapide pour un site statique et workflow Git simple.

1. Creer un repo Git avec ce dossier.
2. Importer le repo dans Vercel.
3. Framework preset: `Other`.
4. Build command: vide.
5. Output directory: `.`
6. Deploy.

### Alternative: Netlify

1. Importer le repo.
2. Build command: vide.
3. Publish directory: `.`
4. Deploy.
